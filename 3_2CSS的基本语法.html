3.2.1<style>和<link>标签

这是<style>标签

使用CSS的最佳实践是使用<link>标记将其保存在外部文件中。但是，在开始时，只需将其直接放入正在编辑的文档中就更简单了。

要将CSS直接放置到HTML文档中，我们使用<style>标记。此标记可以出现在HTML文档中的任何位置，但是，最常见的做法是将其放置在<head>部分。例如：

Example HTML 
  1. <head>
  2.   <meta charset="UTF-8">
  3.   <title>style and link tags
  4.   <style>
  5.   /*CSS将进入此区域*/
  6.   </style>
  7. </header>

这是<linl>标签

虽然<style>很方便，但更好的做法是将CSS放在一个单独的文件中。使用单独文件的一个关键优势是，CSS样式可以在不同的用户之间轻松重复使用。html页面。许多作者进一步将CSS划分为不同的文件（例如：一个用于文本样式，另一个用于布局）。

只需将CSS放在一个单独的文件中。此文件不需要任何HTML标记（即不需要标记）。使用。css文件扩展名，并使用<link>标记将其绑定。标签必须出现在<head>部分。按照惯例，css文件保存在一个名为css的目录中。

将此链接用作模板：

Example HTML 
  1. <link rel="stylesheet" href="css/my_style.css">

下面是一个示例HTML文档。

Example HTML 
  1. <head>
  2.   <meta charset="UTF-8">
  3.   <title>style and link tags</title>
  4.   <link rel="stylesheet" href="css/my_style.css">
  5. </head>

3.2.2规则：选择器和声明

最简单的说，CSS只是一个规则列表。每个规则由一个选择器和一个声明组成。以下是一个示例：

Selector(选择器）

在上面的例子中，选择器是p。如果选择器没有被任何标点符号固定，则假定它与HTML标记匹配。因此，p选择器将CSS规则应用于文档中的所有标签。
我们将在将来介绍更多的选择器可能性。

Declaration（声明）

CSS规则的声明部分用大括号打开和关闭：{}在它们之间，可以放置任意数量的属性值对。

Properties and values(属性和值)

可以通过CSS设置数百种不同的视觉属性。每个属性都有一系列可以设置的可能值。从语法上讲，属性值对很简单。每对由一个属性组成，后跟一个冒号：后跟一个值，以分号结尾；

font-size: 12px

最佳实践

在上面的示例中，整个CSS规则都写在一行上。当CSS规则的声明只有一个属性时，这种情况并不少见。如果CSS规则有多个属性，那么应该编写为每个属性值对使用一行。例如：

Example CSS 
  1. p{
  2.   font-size: 12px;
  3.   line-height: 15px;
  4.   color: #223344;
  5. }

3.2.3评论

CSS也可以包含“注释”，通过这些注释，您（今天的开发人员）可以给您（明天的另一个开发人员）留下注释和提醒。或其他可能阅读你的CSS的人。

注释以/*开头，必须以*/结尾，可以跨多行。但它们不能嵌套。

Example CSS 
  1. p{
  2.   font-size: 8px; /* client insists small text makes them more 'professional'. */
  3.   /* I hope his idea of 'professional' includes paying on time. */
  4.   line-height: 24px; /* see above */
  5.   /* none of the stuff below is working. I don't know why.
  6.   margin-top: 5%;
  7.   margin-bottom:6%;
  8.   */
  9. }
