2.5.1超链接简介

什么是超链接？

在一个典型的网页中，您可能会看到这样一句话：

了解有关我们产品的 更多信息！

或者类似的：

尝试单击蓝色文本或“立即购买！”，如果你还没有。

超链接是将您带到其他位置的任何文本或图像。这可以是：

  • 另一个网页：例如，位于https://en.wikipedia.org/wiki/Hyperlink
  • 书签（网页的特定部分）：例如，位于https://en.wikipedia.org/wiki/Hyperlink#History
    在前面的关于属性的单元中，我们学习了如何使用id属性作为链接目标。如果页面中有一个分区，则可以使用其id属性（例如历史记录）的值，并按照#历史记录以数字符号开头，链接到该分区
  • 本地链接（链接到同一网页的另一部分）：例如\u标记。asp是属于此网站的网页
  • 电子邮件：例如mailto：authors@example.com

为什么您更喜欢文本链接而不是图像

当涉及到超链接时，尽可能使用文本而不是图像：

  • 图像不像文本那样容易理解或识别。
  • 文本更便于访问。
  • 如果图像中有文本，如“立即购买”按钮，搜索引擎将无法识别图像中的文本。图像的备选文本应传达将启动的操作（图像的目的，例如“立即购买”），而不是图像的描述（“按钮”）。

最好的做法

  • 将超链接应用于短短语。在整个段落周围使用链接标记是不常见的。
  • 使链接短语有意义。避免使用“单击此处”或“阅读更多”之类的短语当您可以围绕短语“获取帮助”创建链接时，单击此处获取帮助是多余的。该链接表示应单击该链接。
  • 不要使用短链接文本。如果在一个单词或字符上使用“蓝色”超链接，很容易错过，用户很难使用触摸屏单击。
  • 外观-链接在大多数浏览器中都有默认外观，蓝色和下划线。确保页面中的其他文本没有下划线，以避免混淆用户。他们可能会对点击他们认为是链接的文本感到沮丧。
  • 如果您选择有图像链接，它应该有描述链接目的的替代文本，而不是使用的图像-描述目标链接。
    Example HTML 
    1. [a href="https://www.w3.org/Style/Examples/007/leaders"]
    2. [img src="table-th.png" alt="CSS tips related to dot leaders"]
    3. [/a]

锚元素:[a]

html中的hyperlink标记只是[a>]，它被称为锚元素。下面是它的使用方法：

Example HTML 
  1. [a href="https://en.wikipedia.org/wiki/Hyperlink"]Click here[/a] to go to the Wikipedia Hyperlink page.
Example Result Click here to go to the Wikipedia Hyperlink page.

注意：锚元素不应与[link]标记相混淆。[link]标签用于定义文档和外部资源（如外部样式表）之间的链接。在下一章中，您将了解有关标记的更多信息。

超链接的状态

如果尚未单击该链接，则该链接将为蓝色并带下划线。现在，单击链接，您将看到已访问的链接看起来是紫色的，并带有下划线。除了未访问和已访问的链接之外，还有一个状态称为活动链接。当用户单击链接时，该链接变为活动链接。在此期间，链接将为红色并加下划线。

  • 未访问：蓝色+下划线
  • 访问：紫色+下划线
  • 活动：红色+下划线

使用方法

[a]标签可以围绕文本或图像。

Example HTML 
  1. [!--超链接中的文本--]
  2. [a href="https://www.qwant.com/"]如果你点击我, 我会带你去qwant.com。[/a]
  3. [!--超链接中的段落--]
  4. [a href="https://www.qwant.com/"][p]如果你点击我, 我会带你去qwant.com。[/p][/a]
  5. [!--超链接中的图像--]
  6. [a href="https://www.qwant.com/"][img src="images/qwant-image.png" alt="Image navigating to Qwant"[/a]

您甚至可以使用anchor元素在网页的contacts部分下添加电子邮件地址。

Example HTML 
  1. [p]Feedback:[a href="mailto:authors@example.com?Subject=Hello"]使用SubSec向作者发送邮件[/a][/p]
Example Result

Feedback: Send mail to authors with subject

不过，您可能应该避免对电子邮件地址使用超链接：

  • 电子邮件地址通常在用户计算机上的默认电子邮件客户端上打开，用户可能不会使用该客户端。这违背了整个目的。
  • 在网页中直接以文本或超链接形式提供您的电子邮件地址将显示您的电子邮件地址。机器人程序或垃圾邮件爬虫可用于搜索Web以获取垃圾邮件的电子邮件地址。如果可能的话，你应该避免这种情况，而应该使用联系方式。

2.5.3属性：media和download

媒体属性:media

HTML5中引入了媒体属性。它用于指定在href中链接到的URL针对哪种媒体或设备进行了优化。URL可以针对投影仪、语音合成器或打印页面等特殊设备。如果您希望满足URL指向特定设备类型的目标文档的需要，这将非常有用。

假设您有一个分散着多个图像的页面，并且希望在手持设备上显示此页面。众所周知，手持设备的屏幕很小，带宽有限。我们希望在小屏幕上更好地对齐页面，并减小图像的大小。因此，media属性允许我们告诉anchor元素该页面是针对手持设备的。可以通过为属性提供值来实现这一点。此值可以是任何有效的媒体查询，并且是设备类型和媒体呈现值的组合。

让我们看另一个例子——您可以创建一个指向内容密集的长页面的打印链接，该链接将重定向到相同页面的打印版本。您希望将此打印版本格式化为一页，非常适合打印，分辨率为250 dpi。下面是HTML5代码的样子：

Example HTML 
  1. [a href="https://en.wikipedia.org/wiki/Media_queries?output=print" media="print and (resolution:250dpi)"]打印关于媒体查询的wiki页面[/a]
Example Result Print wiki page about media queries
