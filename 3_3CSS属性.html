3.3.1常见CSS属性

有数百个CSS属性供您使用。完整的列表可以在 W3C网站 上找到（也可以参见 MDN网站上的CSS参考页 ）。

下面我们收集了一个更易于管理的最有用和常见CSS属性列表：font-size字体大小、line-height行高、text-align文本对齐、text-decoration文本装饰、font-weight字体重量、font-style字体样式和font-family字体系列。

font-size 字体大小

字体大小可用于调整标记文本的大小。字体大小的值有两部分：数字和单位。一些最常见的单位是：px、em、%、vh。例如：

EXample-CSS 
p { font-size: 18px; }
q { font-size: .8em; }
blockquote { font-size: 10vh; }
